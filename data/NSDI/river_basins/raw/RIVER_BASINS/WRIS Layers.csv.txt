Below is a list of Web Map Services (WMS), their URL endpoints and their layer names with corresponding data that the layers link to
To get detailed file containing layer metadata associated with each WMS, append "?SERVICE=WMS&VERSION=1.1.1&REQUEST=GetCapabilities" to each WMS URL endpoint

I have only listed layers for WMSes of my interest.
You can search for the associated file in the WMS folder and locate names for layers of your interest by searching for <Name> tags
